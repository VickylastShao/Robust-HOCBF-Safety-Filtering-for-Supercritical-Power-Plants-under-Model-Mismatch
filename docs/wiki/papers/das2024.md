## One-Line Summary

Proposes an uncertainty/disturbance estimator for control-affine nonlinear systems with state- and input-dependent unmodeled dynamics, and uses it in two frameworks---matched uncertainty compensation (Method 1) and robustified CBF/HOCBF constraint reformulation (Method 2)---to guarantee robust safety via convex QPs.

## Problem Setting

The paper considers a nonlinear control-affine system whose true (actual) dynamics are:

$$\dot{x} = f(x) + g(x)u, \quad x \in \mathcal{X} \subset \mathbb{R}^n, \quad u \in \mathcal{U} \subset \mathbb{R}^m$$

Only a nominal model $\hat{f}(x)$, $\hat{g}(x)$ is available for controller design. Adding and subtracting the nominal model reveals the uncertain system:

$$\dot{x} = \hat{f}(x) + \hat{g}(x)u + \underbrace{(f(x) - \hat{f}(x))}_{\Delta_f(x)} + \underbrace{(g(x) - \hat{g}(x))}_{\Delta_g(x)} u$$

The lumped uncertainty is defined as:

$$\Delta(x, u) := \Delta_f(x) + \Delta_g(x)u$$

so the uncertain system takes the compact form:

$$\dot{x} = \hat{f}(x) + \hat{g}(x)u + \Delta(x, u)$$

**Key assumptions on the uncertainty:**

- **Assumption 1:** $\Delta(x,u)$ is locally Lipschitz continuous for all $t \geq 0$ with Lipschitz constant $\delta_L$, and bounded by $\|\Delta(x,u)\| \leq \delta_b$ for all $(x,u) \in \mathcal{X} \times \mathcal{U}$, where $\delta_L, \delta_b \in \mathbb{R}_+$ are known constants.

- **Assumption 2 (for Method 1):** The input relative degree (IRD) equals the disturbance relative degree (DRD), i.e., matched uncertainty. This means $\Delta(x,u)$ and $u$ appear on the same output channel.

**Control objective:** Given a potentially unsafe nominal controller $k_d(x)$ and a CBF $h(x)$ defining a safe set $\mathcal{C} = \{x \in \mathcal{X} : h(x) \geq 0\}$, find a minimally-modified safe control input that guarantees forward invariance of $\mathcal{C}$ despite the unmodeled dynamics $\Delta(x,u)$.

## Key Contributions

1. **Novel uncertainty/disturbance estimator** for MIMO control-affine systems with state- and input-dependent uncertainties, extending prior disturbance observers (which were limited to state-dependent external disturbances) to the general $\Delta(x,u)$ case.
2. **Theoretical error and output bounds** (Lemma 3) that provide explicit time-dependent convergence guarantees for the estimation error $\|e(t)\|$ and the estimated output $\|\hat{\Delta}(t)\|$, expressed in terms of $\delta_b$, $\delta_L$, and the estimator gain $\Lambda$.
3. **Method 1 -- Uncertainty compensation for matched uncertainty:** Uses the estimator to actively reject matched uncertainty via input augmentation $u = \tilde{u} - Q(x)\hat{\Delta}$, modifies the CBF to $h_V(x,e) = h(x) - \sigma_V V_e(e)$, and derives a robust safe set $\mathcal{C}_V \subseteq \mathcal{C}$ with a modified CBF-QP.
4. **Method 2 -- Robustified CBF constraint with estimator + error bound:** Replaces the unknown $\Delta(x,u)$ in the CBF time derivative with the estimate $\hat{\Delta}$ and subtracts the error bound $\left\|\frac{\partial h}{\partial x}\right\|\|e(t)\|$ to robustify the constraint. Applies to both standard CBFs (Theorem 3) and HOCBFs (Corollary 1).
5. **HOCBF extension (Corollary 1):** Derives the robustified high-order CBF constraint that incorporates the uncertainty estimate and error bound for safety constraints with relative degree $m > 1$.

## Core Formulation

### Uncertainty Estimator

The proposed estimator has the structure:

$$\hat{\Delta}(t) = \Lambda x - \xi(t) \tag{23}$$

$$\dot{\xi}(t) = \Lambda\bigl(\hat{f}(x) + \hat{g}(x)u + \hat{\Delta}(t)\bigr) \tag{24}$$

where $\xi \in \mathbb{R}^n$ is an auxiliary state vector and $0 \prec \Lambda \in \mathbb{R}^{n \times n}$ is a diagonal positive definite estimator design matrix, i.e., $\Lambda = \mathrm{diag}(\lambda_1, \ldots, \lambda_n)$. Initial values are set to $\hat{\Delta}(0) = 0$ and $\xi(0) = \Lambda x(0)$.

### Estimation Error Dynamics

Defining the estimation error as $e(t) = \Delta(x,u) - \hat{\Delta}(t)$ with $e(t_0) = e_0$, the error dynamics are:

$$\dot{e}(t) = \dot{\Delta}(t) - \Lambda e(t), \quad e(t_0) = e_0 \tag{26}$$

**Lemma 1 (ISS of error dynamics):** The error dynamics are input-to-state stable (ISS) around $e = 0$:

$$\dot{V}_e(e, \dot{e}) \leq -2\mu_e V_e + \gamma(\|\delta_L\|)$$

where $V_e(e) = \frac{1}{2}e^T e$, $\mu_e = \frac{\lambda_{\min}}{4}$, and $\gamma(\|\delta_L\|) = \frac{\delta_L^2}{2\lambda_{\min}}$.

**Lemma 3 (Explicit bounds):** Under Assumption 1 with the Lyapunov equation solution $P$ satisfying $P\Lambda + \Lambda^T P = -I$:

$$\|e(t)\| \leq P(\delta_b - 2\delta_L\|P\|)e^{-\frac{t}{2\|P\|}} + 2\|P\|\delta_L \tag{32}$$

$$\|\hat{\Delta}(t)\| \leq 2P\delta_b\|\Lambda\|\|P\|\bigl(1 - e^{-\frac{t}{2\|P\|}}\bigr) + 2\|P\|\delta_L \tag{33}$$

### Method 1: Uncertainty Compensation for Matched Uncertainty

When IRD = DRD (matched uncertainty), find a matrix $Q \in \mathbb{R}^{m \times n}$ satisfying:

$$\hat{g}(x)Q(x)\Delta(x,u) = \Delta(x,u), \quad \forall t \geq 0 \tag{36}$$

If $\mathrm{rank}(\hat{g}(x)) = m$ and the pseudo-inverse exists, then:

$$Q(x) := (\hat{g}(x)^T\hat{g}(x))^{-1}\hat{g}^T(x) \tag{38}$$

The augmented control law is:

$$u = \tilde{u} - Q(x)\hat{\Delta} \tag{39}$$

where $\tilde{u} = k(x)$ is the nominal controller and $u_{\hat{\Delta}} = Q(x)\hat{\Delta}$ is the uncertainty attenuation term. The closed-loop dynamics become:

$$\dot{x} = \hat{f}(x) + \hat{g}(x)\tilde{u} + e(t) \tag{40}$$

A modified CBF is introduced to provide robustness against estimation error:

$$h_V(x, e) = h(x) - \sigma_V V_e(e), \quad \sigma_V \in \mathbb{R}_+ \tag{41}$$

$$\mathcal{C}_V := \{x \in \mathcal{X} \mid h_V(x,e) \geq 0\} \tag{42}$$

**Theorem 2:** If there exists $\mu_h \in \mathbb{R}_+$ such that $\mathcal{D} := 4\sigma_V\mu_e - 2\sigma_V\mu_h > 0$ and the condition holds:

$$L_{\hat{f}}h(x) + L_{\hat{g}}h(x)\tilde{u} \geq -\mu_h h(x) + \frac{\left\|\frac{\partial h}{\partial x}\right\|^2}{\mathcal{D}} + \sigma_V\gamma(\|\delta_L\|) \tag{44}$$

then $x_0 \in \mathcal{C} \Longrightarrow x(t) \in \mathcal{C}$.

### Method 2: Robustify CBF Constraint with Estimator + Error Bound

Instead of compensating the uncertainty in the input channel, Method 2 directly robustifies the CBF constraint. Starting from the time derivative of $h(x)$ for the uncertain system:

$$\dot{h}(x,u) = L_{\hat{f}}h(x) + L_{\hat{g}}h(x)u + \frac{\partial h}{\partial x}\Delta(x,u) \tag{46}$$

Replacing $\Delta = \hat{\Delta} + e$ and lower-bounding the error term yields:

$$\dot{h}(x,u) \geq L_{\hat{f}}h(x) + L_{\hat{g}}h(x)u + \frac{\partial h}{\partial x}\hat{\Delta} - \left\|\frac{\partial h}{\partial x}\right\|\|e(t)\| \tag{48}$$

**Theorem 3:** If $\frac{\partial h}{\partial x} \neq 0$, $\forall x \in \mathcal{C}$ and there exists $\alpha \in \mathcal{K}_{\infty,e}$ such that $\forall x \in \mathcal{C}$:

$$\sup_{u \in \mathcal{U}} \left[\frac{\partial h}{\partial x}\bigl(\hat{f}(x) + \hat{g}(x)u + \hat{\Delta}\bigr) - \left\|\frac{\partial h}{\partial x}\right\|\|e(t)\|\right] \geq -\alpha(h(x)) \tag{47}$$

then $x_0 \in \mathcal{C} \Longrightarrow x(t) \in \mathcal{C}$.

### HOCBF Extension (Corollary 1)

For a valid HOCBF $h(x)$ with relative degree $m$ and a sequence of functions:

$$\varphi_0(x) := h(x), \quad \varphi_i(x) := \dot{\varphi}_{i-1}(x) + \alpha_i(\varphi_{i-1}(x)) \tag{18}$$

assuming IRD = DRD = $m$, the robustified HOCBF constraint is:

$$L_{\hat{f}}^m h(x) + L_{\hat{g}}L_{\hat{f}}^{m-1} h(x)\cdot u + \frac{\partial L_{\hat{f}}^{m-1} h(x)}{\partial x}\hat{\Delta} - \left\|\frac{\partial L_{\hat{f}}^{m-1} h(x)}{\partial x}\right\|\|e(t)\| + O(h(x)) \geq -\alpha_m(\psi_{m-1}) \tag{50}$$

where $O(h(x)) := \sum_{i=1}^{m-1} L_f(\alpha_{m-i} \circ \varphi_{m-i-1})$ collects the lower-order terms, and $\psi_{m-1}$ denotes the $(m-1)$-th extended class-$\mathcal{K}$ function output.

## Theoretical Results

1. **ISS of estimation error (Lemma 1):** The error dynamics $\dot{e} = \dot{\Delta} - \Lambda e$ are ISS around $e = 0$, ensuring bounded estimation error under bounded uncertainty rate of change.

2. **Explicit convergence bounds (Lemma 3):** Both $\|e(t)\|$ and $\|\hat{\Delta}(t)\|$ converge exponentially to steady-state bounds that depend on $\delta_L$ (Lipschitz constant of $\Delta$), $\delta_b$ (upper bound on $\Delta$), and the estimator eigenvalues via $\|P\|$. The steady-state error bound is $2\|P\|\delta_L$.

3. **Safety with uncertainty compensation (Theorem 2, Method 1):** Under the matching condition and with the modified CBF $h_V(x,e) = h(x) - \sigma_V V_e(e)$, if $\mathcal{D} = 4\sigma_V\mu_e - 2\sigma_V\mu_h > 0$ and the augmented CBF constraint (44) holds, then the original safe set $\mathcal{C}$ is forward invariant.

4. **Safety with robustified constraint (Theorem 3, Method 2):** If the robustified CBF constraint (47) holds---which uses $\hat{\Delta}$ for the estimated part and $\left\|\frac{\partial h}{\partial x}\right\|\|e(t)\|$ as a worst-case error margin---then the safe set $\mathcal{C}$ is forward invariant.

5. **HOCBF safety (Corollary 1):** The robustification extends to high relative degree constraints, with the key term being $\frac{\partial L_{\hat{f}}^{m-1}h}{\partial x}\hat{\Delta} - \left\|\frac{\partial L_{\hat{f}}^{m-1}h}{\partial x}\right\|\|e(t)\|$ in the $m$-th order CBF constraint.

6. **Alternative Method 1 QP (Corollary 2):** An alternative robust CBF-QP for Method 1 explicitly uses error dynamics in the constraint:

$$\sup_{\tilde{u} \in \mathcal{U}} \left[\frac{\partial h}{\partial x}(\hat{f}(x) + \hat{g}(x)\tilde{u}) - \left\|\frac{\partial h}{\partial x}\right\|\|e(t)\|\right] \geq -\alpha(h(x))$$

## Algorithm / Implementation

### Estimator Design

1. Choose the diagonal gain matrix $\Lambda = \mathrm{diag}(\lambda_1, \ldots, \lambda_n)$ with $\lambda_i > 0$. Larger gains reduce estimation error but may amplify noise sensitivity.
2. Initialize $\hat{\Delta}(0) = 0$ and $\xi(0) = \Lambda x(0)$.
3. At each time step, update $\xi$ via (24) and compute $\hat{\Delta} = \Lambda x - \xi$.
4. Determine $\delta_L$ (Lipschitz constant) and $\delta_b$ (upper bound) for the uncertainty from domain knowledge or empirical data.

### Method 1 QP (Matched Uncertainty)

$$\tilde{u}^*(x) = \arg\min_{\tilde{u} \in \mathcal{U}} \|\tilde{u} - k_d(x)\|^2 \quad \text{s.t.} \quad L_{\hat{f}}h(x) + L_{\hat{g}}h(x)\tilde{u} \geq S(x)$$

where $S(x) = -\mu_h h(x) + \frac{\left\|\frac{\partial h}{\partial x}\right\|^2}{\mathcal{D}} + \sigma_V\gamma(\|\delta_L\|)$ and $\mathcal{D} = 4\sigma_V\mu_e - 2\sigma_V\mu_h > 0$.

The applied control is $u = \tilde{u}^* - Q(x)\hat{\Delta}$.

### Method 2 QP (General, including HOCBF)

$$u^*(x) = \arg\min_{u \in \mathcal{U}} \|u - k_d(x)\|^2$$

subject to the robustified constraint (for relative degree 1):

$$L_{\hat{f}}h(x) + L_{\hat{g}}h(x)u + \frac{\partial h}{\partial x}\hat{\Delta} - \left\|\frac{\partial h}{\partial x}\right\|\|e(t)\| \geq -\alpha(h(x))$$

or for relative degree $m$ (HOCBF, Corollary 1):

$$L_{\hat{f}}^m h(x) + L_{\hat{g}}L_{\hat{f}}^{m-1} h(x)\cdot u + \frac{\partial L_{\hat{f}}^{m-1} h(x)}{\partial x}\hat{\Delta} - \left\|\frac{\partial L_{\hat{f}}^{m-1} h(x)}{\partial x}\right\|\|e(t)\| + O(h(x)) \geq -\alpha_m(\psi_{m-1})$$

Both QPs are convex (linear constraints, quadratic objectives) and can be solved efficiently at runtime.

## Limitations

1. **Matching condition for Method 1:** Method 1 requires IRD = DRD, i.e., the uncertainty must appear on the same channel as the control input. This excludes mismatched uncertainties common in many underactuated systems. Finding a valid $Q(x)$ matrix for general MIMO nonlinear systems remains an open problem.

2. **Bounded uncertainty assumption:** Assumption 1 requires known bounds $\delta_b$ (upper bound on $\|\Delta\|$) and $\delta_L$ (Lipschitz constant of $\Delta$). In practice, these constants may be difficult to estimate accurately, and over-estimation leads to conservativeness while under-estimation voids safety guarantees.

3. **Lipschitz continuity of $\Delta$:** The uncertainty must be locally Lipschitz continuous. Discontinuities or non-smooth uncertainties (e.g., hybrid dynamics, contact forces) are not covered by the theory.

4. **Conservativeness of error bound:** The term $\left\|\frac{\partial h}{\partial x}\right\|\|e(t)\|$ in Method 2 provides a worst-case lower bound that can be conservative, especially when the estimation error direction aligns favorably. The steady-state error bound $2\|P\|\delta_L$ may also be loose.

5. **Tuning sensitivity in Method 1:** The parameter $\sigma_V$ controls the shrinkage of the safe set $\mathcal{C}_V \subseteq \mathcal{C}$. Larger $\sigma_V$ provides more robustness but shrinks the safe set, potentially leading to infeasibility. The condition $\mathcal{D} = 4\sigma_V\mu_e - 2\sigma_V\mu_h > 0$ constrains the admissible values.

6. **Full-state measurement:** The estimator assumes full-state observability. Partial state measurements would require a state observer, adding complexity and potential estimation error.

7. **No learning component:** The estimator is purely model-based with fixed gains. It does not improve its estimates from data over time, and the bounds $\delta_L$, $\delta_b$ must be pre-specified.

## Relevance to RoCBF-Net

**Phase 2 relevance:** This paper provides the core structural template for the RoCBF-Net robustification approach.

### What we adopt

- **Uncertainty estimator structure:** The estimator form $\hat{\Delta}(t) = \Lambda x - \xi(t)$ with $\dot{\xi} = \Lambda(\hat{f} + \hat{g}u + \hat{\Delta})$ provides a principled real-time uncertainty estimation framework. The ISS analysis and error bounding methodology directly inform our safety guarantees.
- **HOCBF robustification pattern (Corollary 1):** The structure of the robustified constraint---using an uncertainty estimate plus a worst-case error margin---is the template we follow:

$$L_{\hat{f}}^m h + L_{\hat{g}}L_{\hat{f}}^{m-1} h \cdot u + \frac{\partial L_{\hat{f}}^{m-1}h}{\partial x}\hat{\Delta} - \left\|\frac{\partial L_{\hat{f}}^{m-1}h}{\partial x}\right\|\|e(t)\| + O(h) \geq -\alpha_m(\psi_{m-1})$$

- **Method 2 framework:** The idea of robustifying the CBF constraint (rather than compensating in the input channel) is our primary approach, since it does not require the matching condition and naturally extends to HOCBFs.

### What we modify

- **Replace $\hat{\Delta}$ with GP posterior mean:** Instead of the deterministic observer $\hat{\Delta}(t) = \Lambda x - \xi(t)$, we use the Gaussian process posterior mean $\mu_{\mathrm{GP}}(x, u)$ as the uncertainty estimate. This provides a data-driven, non-parametric estimator that improves with more observations and does not require pre-specified $\delta_L$, $\delta_b$.
- **Replace error bound $\|e(t)\|$ with GP posterior variance:** Instead of the deterministic error bound $\|e(t)\| \leq 2\|P\|\delta_L$ (which requires known $\delta_L$), we use a calibrated confidence interval based on the GP posterior standard deviation $\sigma_{\mathrm{GP}}(x, u)$. Specifically, we replace $\left\|\frac{\partial L_{\hat{f}}^{m-1}h}{\partial x}\right\|\|e(t)\|$ with a term proportional to $\left\|\frac{\partial L_{\hat{f}}^{m-1}h}{\partial x}\right\|\beta^{1/2}\sigma_{\mathrm{GP}}(x, u)$, where $\beta$ is a scaling factor derived from GP regret bounds to ensure high-probability safety.
- **Remove dependence on known $\delta_L$, $\delta_b$:** The GP framework automatically learns the uncertainty characteristics from data, eliminating the need for the practitioner to pre-specify Lipschitz constants and upper bounds---a key practical limitation of the original formulation acknowledged by the authors themselves in their future work section.
