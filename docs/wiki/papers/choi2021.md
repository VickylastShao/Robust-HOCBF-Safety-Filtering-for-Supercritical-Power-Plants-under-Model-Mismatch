## One-Line Summary

The paper introduces the Control Barrier-Value Function (CBVF), which merges HJ reachability value functions with CBFs to recover the maximal robust safe set while enabling less conservative online control via a QP formulation.

## Problem Setting

- **System model**: Continuous-time time-invariant system with disturbance:
  $$\dot{x}(s) = f(x(s), u(s), d(s)), \quad s \in [t, 0], \quad x(t) = x$$
  where $u \in \mathcal{U} \subset \mathbb{R}^m$ is the control input, $d \in \mathcal{D} \subset \mathbb{R}^w$ is the disturbance, $\mathcal{U}$ and $\mathcal{D}$ are compact and convex, and $f$ is Lipschitz continuous in state and bounded.
- **Disturbance model**: Bounded disturbance $d \in \mathcal{D}$ following nonanticipative strategies $\xi_d \in \Xi_{[t,0]}$ (i.e., disturbance can react to control but is nonanticipative — a differential game setting).
- **Control objective**:
  1. Compute the viability kernel $\mathcal{S}(t)$ for a constraint set $\mathcal{L} = \{x : l(x) \geq 0\}$ — the set of states from which there exists an admissible control that keeps the system safe under worst-case disturbance.
  2. Compute a robust safe control signal for each $x \in \mathcal{S}(t)$.

## Key Contributions

1. **Definition of the Robust CBVF** — a new value function $B_\gamma$ that blends the CBF discount-rate structure ($\alpha(z) = \gamma z$) into the HJ reachability formulation, recovering the maximal robust safe set (viability kernel) rather than a conservative under-approximation as typical CBFs do.
2. **Verification that $B_\gamma$ is the unique viscosity solution** of a novel Hamilton-Jacobi-Isaacs Variational Inequality (CBVF-VI), providing a constructive computational method for the CBVF.
3. **Proof that the zero-superlevel set of $B_\gamma$ equals the viability kernel** ($\mathcal{C}_\gamma(t) = \mathcal{S}(t)$), in contrast to hand-crafted CBFs whose safe sets are only subsets of the maximal invariant set.
4. **The Robust CBVF-QP** — an online quadratic program for control-affine systems whose solution is always optimal with respect to $B_\gamma$ and is feasible everywhere the gradient exists, serving as a smoother and less conservative safety filter than the original HJ optimal policy.
5. **Demonstration** that the CBVF-based controller is less conservative than the standard HJ reachability optimal policy and more permissive than CBF-QP with infinite-horizon safety, on double-integrator and Dubins car examples.

## Core Formulation

### CBVF Definition

The Robust Control Barrier-Value Function is defined as:

$$B_\gamma(x, t) := \min_{\xi_d \in \Xi_{[t,0]}} \max_{u \in \mathcal{U}_{[t,0]}} \min_{s \in [t,0]} e^{\gamma(s-t)} l(x(s))$$

where $x(\cdot)$ solves the dynamics for $(x, t, u, \xi_d[u])$, with terminal condition $B_\gamma(x, 0) = l(x)$, and $\gamma \geq 0$ is a discount-rate parameter.

- When $\gamma = 0$: $B_0$ reduces to the standard HJ reachability value function $V(x,t)$.
- The exponential weighting $e^{\gamma(s-t)}$ means that the safety target function decays backward in time at rate $\gamma$, which is precisely the CBF property that $B$ should not decay faster than $\dot{B} = -\gamma B$.

### Merging HJ Reachability and CBFs

The key insight is that the CBF condition $\max_u D_x B(x) \cdot f(x,u) \geq -\alpha(B(x))$ with $\alpha(z) = \gamma z$ is strictly less restrictive than the HJ optimal policy condition $\min_d D_x V(x,s) \cdot f(x,u,d) \geq 0$. The CBVF formulation embeds this less restrictive structure directly into the value function definition via the exponential discount.

### HJI Variational Inequality (CBVF-VI)

$B_\gamma$ is the unique Lipschitz continuous viscosity solution of:

$$0 = \min\left\{l(x) - B_\gamma(x,t),\; D_t B_\gamma(x,t) + \max_{u \in \mathcal{U}} \min_{d \in \mathcal{D}} D_x B_\gamma(x,t) \cdot f(x,u,d) + \gamma B_\gamma(x,t)\right\}$$

with terminal condition $B_\gamma(x,0) = l(x)$.

- Compare to the standard HJI-VI for $V$: the only difference is the addition of $+\gamma B_\gamma(x,t)$ in the Hamiltonian, which comes from the CBF discount rate.
- The $\min\{\cdot,\cdot\}$ structure acts as a constraint: if $B_\gamma(x,t) < l(x)$, the Hamiltonian term must be zero; if $B_\gamma(x,t) = l(x)$, the Hamiltonian term must be non-negative.

### CBVF-QP Formulation

For control-affine systems with disturbance:
$$\dot{x}(s) = p(x(s)) + q(x(s))u(s) + r(x(s))d(s)$$

The Robust CBVF-QP is:

$$\pi_{QP}(x,t) = \arg\min_{u \in \mathcal{U}} (u - u_{ref})^T(u - u_{ref})$$

subject to:
$$a(x,t) + D_x B_\gamma(x,t) \cdot q(x)u + \gamma B_\gamma(x,t) \geq 0$$

where $a(x,t) = D_t B_\gamma(x,t) + D_x B_\gamma(x,t) \cdot p(x) + \min_{d \in \mathcal{D}} D_x B_\gamma(x,t) \cdot r(x)d$.

### Robustness to Disturbances

- The formulation uses a minimax (differential game) structure: the disturbance acts adversarially (through nonanticipative strategies), while the controller optimizes against the worst case.
- The $\min_d$ term in the QP constraint ensures the safety filter accounts for the worst-case disturbance at every point.
- Demonstrated empirically: trajectories remain within the safe set even under worst-case bounded disturbances.

## Theoretical Results

- **$B_\gamma$ as viscosity solution** (Theorem 3): $B_\gamma$ is the unique Lipschitz continuous viscosity solution of the CBVF-VI. This provides both existence and uniqueness, and justifies numerical computation via dynamic programming on a grid.

- **Safety guarantee** (Proposition 2): For each $t \leq 0$, $\mathcal{C}_\gamma(t) := \{x : B_\gamma(x,t) \geq 0\} = \mathcal{S}(t)$. That is, the zero-superlevel set of the CBVF exactly equals the viability kernel — the maximal robust safe set. This is a strict improvement over hand-crafted CBFs whose zero-superlevel sets are only subsets of $\mathcal{S}_\infty$.

- **Optimality of the CBVF-QP** (Proposition 3): For control-affine systems with linear control bounds, the Robust CBVF-QP is feasible everywhere $D_x B_\gamma$ exists, and its solution is always an optimal policy with respect to $B_\gamma$.

- **Less conservative than HJ optimal policy**: When $B_\gamma(x,t) = l(x)$ (the "non-critical" case), the CBVF allows $\dot{l} \geq -\gamma l$, permitting the state to approach the safety boundary, whereas the standard HJ policy requires $\dot{l} \geq 0$ (never approaching the boundary). This is the key source of reduced conservativeness.

- **Dynamic programming principle** (Theorem 2): The CBVF satisfies a Bellman-type optimality condition with exponential discount, which underpins the viscosity solution proof.

## Algorithm / Implementation

### Offline: HJ Value Function Computation

1. Define the safety target function $l(x)$ and the constraint set $\mathcal{L} = \{x : l(x) \geq 0\}$.
2. Choose the discount parameter $\gamma \geq 0$.
3. Solve the CBVF-VI backward in time using dynamic programming on a state-space grid:
   - Terminal condition: $B_\gamma(x,0) = l(x)$.
   - At each time step, update $B_\gamma$ via the variational inequality on the grid.
   - Standard HJ computation tools (e.g., Level Set Toolbox, helperOC) can be adapted by modifying the Hamiltonian to include the $+\gamma B_\gamma$ term.
4. Store $B_\gamma(x,t)$ and its spatial gradient $D_x B_\gamma(x,t)$ for online use.

### Online: CBVF-QP Safety Filter

1. At each time step with state $x$ and time $t$:
   - Look up $B_\gamma(x,t)$, $D_t B_\gamma(x,t)$, $D_x B_\gamma(x,t)$ from the precomputed grid (with interpolation).
   - Compute $a(x,t) = D_t B_\gamma + D_x B_\gamma \cdot p(x) + \min_d D_x B_\gamma \cdot r(x)d$.
   - Solve the QP: minimize $\|u - u_{ref}\|^2$ subject to $a(x,t) + D_x B_\gamma \cdot q(x)u + \gamma B_\gamma \geq 0$, $u \in \mathcal{U}$.
2. Apply the QP solution $u^*$ to the system.

### Computational Complexity Notes

- The offline computation scales exponentially with state dimension $n$ (curse of dimensionality of HJ methods).
- The online QP is cheap (polynomial in control dimension $m$) and suitable for real-time implementation.
- Parameter $\gamma$ trades off conservativeness vs. safety margin: larger $\gamma$ allows more aggressive control but with tighter safety margins near the boundary.

## Limitations

- **Curse of dimensionality**: The offline HJ computation requires solving the CBVF-VI on a grid, which scales exponentially with state dimension $n$. The paper notes that scalability enhancements from the reachability community (e.g., DeepReach, decomposition) can help, but the fundamental scaling issue remains.
- **Conservatism from worst-case disturbance**: The minimax formulation ensures robustness to the worst-case disturbance, which can be conservative when disturbances are not adversarial.
- **Finite-time horizon**: The CBVF provides safety guarantees for a finite time horizon $[t, 0]$. Extending to infinite horizon ($t \to -\infty$) recovers $V_\infty$ but loses the benefit of the $\gamma$ discount.
- **Differentiability**: The QP requires $D_x B_\gamma$ to exist. At points of non-differentiability, super/sub-differentials must be used (Remark 4), which complicates implementation.
- **Grid-based gradient errors**: Numerical gradients from the HJ grid can be inaccurate, especially near non-differentiable points, potentially degrading QP performance.
- **Conservative for non-adversarial settings**: The differential game formulation assumes the worst, which is overly conservative when disturbances are stochastic rather than adversarial.

## Relevance to RoCBF-Net

This paper presents a fundamentally different approach to robust safety than RoCBF-Net. Key contrasts:

- **HJ-based vs. GP-based robustness**: The CBVF obtains robustness via the minimax (differential game) structure of HJ reachability, computing worst-case disturbance effects on a grid. RoCBF-Net uses Gaussian Processes to model uncertainty, which is data-driven and does not require a priori bounded disturbance sets.
- **Curse of dimensionality**: The CBVF's primary limitation is the exponential cost of HJ grid computation, which restricts it to low-dimensional systems ($n \leq 4{-}5$ in practice). RoCBF-Net avoids this entirely by using GP regression instead of grid-based PDE solving, making it applicable to higher-dimensional systems.
- **Constructive CBF**: The CBVF provides a constructive method to obtain a valid CBF (i.e., $B_\gamma$ itself is a valid CBF with $\alpha(z) = \gamma z$), addressing the "hand-crafting" problem in CBF literature. This is relevant as related-work comparison — both approaches address the CBF construction problem but from different angles.
- **Worst-case vs. probabilistic guarantees**: The CBVF provides deterministic worst-case safety guarantees (for bounded disturbances), while RoCBF-Net provides probabilistic safety guarantees (via GP confidence intervals). This is an important distinction for the related-work section.
- **Maximal safe set recovery**: The CBVF recovers the exact viability kernel $\mathcal{S}(t)$, which is the maximal safe set. RoCBF-Net's safe set depends on the GP model and may not be maximal, but it is applicable in settings where the disturbance model is not known a priori.
- **CBVF-QP structure**: The online QP formulation (with $\gamma B_\gamma$ term in the constraint) is structurally similar to the Robust CBF-QP used in RoCBF-Net, but the source of the safety certificate ($B_\gamma$ from HJ vs. $B$ from GP-CBF) differs fundamentally.
